Entity master data — downloads

Same 15 Ghana entities, exported in every format the team uses. All sit in the data/ folder next to this app and are regenerated by python export_entity_data.py in the project.

Format File Use it for Action
CSV (UTF-8 BOM) .csv Open in Excel / Google Sheets. Drop into Salesforce, HubSpot, Pipedrive importer. Download
TSV (tab-separated) .tsv Use when commas inside services/notes break a CSV import. Download
vCard 3.0 .vcf One-tap import to phone contacts (iOS, Android, Outlook). Tagged SolarPro-Beta so they group together. Download
Markdown table .md Paste into Notion, Slack, or an email — renders as a table everywhere. Download
JSON (master, nested) .json Source of truth. Use for engineering integrations and re-builds. Download
JSON (flat, CRM-safe) .flat.json Same data but no arrays in any cell — accepted by CRMs that reject lists. Download
