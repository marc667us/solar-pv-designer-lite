Recalculate rates — {{ project.project_name }} (#{{ project.id }})

Recompute Total Rate and Amount for every item in this project using the current compound rate build-up formula:

Rate = (Supply + Install) × (1 + (OH% + Profit%)/100) × (1 + Cont%/100) × (1 + VAT%/100)

Items saved with the older FLAT-additive formula (rate = direct × (1 + sum of all pcts / 100)) will see their totals increase ~5–10% because the new compound formula correctly nests contingency and VAT on top of the marked-up subtotal — tax-authority defensible (GRA / FIRS / KRA).

Per-item percentages stored in the rate build-up table are not changed by this action — only the recomputed totals are written back.
