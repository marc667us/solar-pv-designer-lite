Calculation Notes & Standards
  • Voltage drop tabulated values: BS 7671:2018 Appendix 4, Tables 4D2B / 4D5B (mV/A/m)
  • Three-phase VD factor: ×0.866 (= √3/2) applied to single-phase tabulated value
  • Temperature correction (Ct): BS 7671 Table 4B2 — reference 30°C
  • Grouping correction (Cg): BS 7671 Table 4B1 — circuits in same enclosure
  • Min Iz = Ib / (Ct × Cg) — current-carrying capacity after derating
  • VD limits: inverter→DB 1.5%; main feeder 2.5%; sub-distribution 3.0%; grid/gen 2.0%
  • Breaker rated at next standard size above Ib × 1.05, must not exceed cable Iz
  • RCCB: 30 mA Type A (BS EN 61008) on all socket-outlet circuits
  • SPD: Type 2 (BS EN 61643) at main DB and any sub-DBs
  • Cable lengths are estimated — verify actual site lengths before finalising
