{# ── Methodology footer ──────────────────────────────────────── #}
Methodology
  1. Sun position via the NOAA Solar Position Algorithm (≈0.1° accuracy) at the site GPS coordinates.
  2. 3D ray-cast shadow projection from each obstruction's bounding geometry onto the panel array plane.
  3. Per-panel intersection via Sutherland-Hodgman convex polygon clipping at 30-minute intervals across the day.
  4. Electrical-string model that applies the correct loss formula per mitigation strategy: None / Bypass diodes (3 substrings per panel + Kirchhoff string penalty) / DC optimisers / Micro-inverters (per-panel MPPT).
  5. 8-bucket conservative pick matching the project's SHADING_FACTORS table; if computed loss falls between bands, the conservative bucket is selected.
  6. Narrative explanation from a Google-ADK LlmAgent whose system prompt carries NREL solar geometry, IEC 61853 string-electrical behaviour, and common site patterns.
